## Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

## Installation

1. Get a free API Key at [https://api.nasa.gov/](https://api.nasa.gov/)
2. Clone the repo
   ```sh
   git clone https://github.com/CsizmadiaLaszlo/ask-nasa.git
   ```
   
3. Enter your API in `config.js`
   ```js
   const API_KEY = 'YOUR API KEY';
   ```
